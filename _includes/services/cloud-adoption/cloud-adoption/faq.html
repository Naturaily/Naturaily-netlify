FAQ

More you need to know about cloud migration

  • 01

    What is hybrid cloud computing?

    A combination of Cloud providers - such as Amazon Web Services, SAP HANA Cloud Platform, VMWare or Salesforce - and private IT infrastructure, including private clouds and even on-premise servers, custom designed for the specific needs of your organization.

  • 02

    What are microservices?

    Microservices are a variant of the service-oriented architecture (SOA), which arrange the application as a set of separate components responsible for specified parts of your business logic. Thanks to microservices-based architecture, you can run your applications more efficiently with containerization and scalable components, improving performance and flexibility.

  • 03

    What is containerization?

    Use containers to run your software more reliably across environments and the cloud, making your applications more lightweight.
