IoT application case study

LV Cloud is an IoT application monitoring and taking care of the local energy grid. The app detects problems with electricity networks even before they occur, helping to prevent major power outages. The LV Cloud application is deployed on the Microsoft Azure Cloud, which provides scalability and flexibility of the infrastructure.

Going into technological details, the LV Cloud solution is based on microservices that are deployed to Microsoft Azure using Kubernetes (an open-source container orchestration platform).
